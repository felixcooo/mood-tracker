## Description
Mood Tracker is a full-stack web application that allows users to log their daily moods.
The app demonstrates full CRUD functionality using a React frontend, an Express API,
and a PostgreSQL database.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: PostgreSQL

## Features
- Create a mood entry
- View all moods
- Delete moods
- RESTful API with Express
- PostgreSQL database integration

## Deployment

The application is structured for deployment on cloud platforms.

- The backend can be deployed using services such as Render or Railway.
- The PostgreSQL database can be hosted using a managed database provider.
- Environment variables would be configured on the hosting platform.
- The frontend can be deployed using Vercel or Netlify.

Due to time constraints, the application is demonstrated locally,
but the project follows best practices for deployment.
